# Cattle Classifier

This project is a cattle classifier that uses machine learning algorithms to classify different breeds of cattle.

## Instructions for Running Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mbyx/cattle_classifier.git
   cd cattle_classifier
   ```

2. **Install and Run**:
   Make sure you have Python and the uv package manager installed, and then run:
   ```bash
   uv run streamlit run main.py
   ```

4. **Access the application**:
   Open your web browser and navigate to `http://localhost:8501`.

## Deployed URL

You can also access the deployed version of the application here: [Cattle Classifier](https://ncai-cattle-classifier.streamlit.app/)
